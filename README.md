# CrispDeck

CrispDeck is an Agentic AI-powered learning content transformation platform that converts academic PDFs and notes into structured flashcards and Anki-compatible learning decks.

## Architecture

PDF
↓
Extraction Agent
↓
Concept Agent
↓
Flashcard Generation Agent
↓
Validation Agent
↓
Deck Builder

## Current Features

- PDF Text Extraction
- AI Concept Extraction
- Flashcard Generation
- Validation Pipeline
- CSV Export

## Tech Stack

- Python
- Gemini API
- PyMuPDF
- Pandas
- GenAnki
